Return a list of ALL movies to the user /movies GET none A JSON object holding data about all the movies
Return data (description, genre, director, image URL, whether it’s featured or not) about a single movie by title /movies/[title] GET None A JSON object holding data about a single movie, like so:,
{
title: 'Pulp Fiction',
description: 'The lives of two mob hitmen, a boxer, a gangster and his wife, and a pair of diner bandits intertwine in four tales of violence and redemption.',
year: 1994,
genres: [crime, drama],
director: 'Quentin Tarantino',
imageURL: '',
}
Return data about a genre (description) by name/title /movies/genres/[name] GET None A JSON object holding data about the specified genre, like so
{
name: 'Science fiction (scifi)',
description: 'Science fiction (once known as scientific romance) is similar to fantasy, except stories in this genre use scientific understanding to explain the universe that it takes place in. It generally includes or is centered on the presumed effects or ramifications of computers or machines; travel through space, time or alternate universes; alien life-forms; genetic engineering; or other such things. The science or technology used may or may not be very thoroughly elaborated on.',
sub-genres: 'Cyberpunk'
}
Return data about a director (bio, birth year, death year) by name /directors/[name] GET None A JSON object holding data about the specified director, like so
{

born in: 'March 27, 1963'
place of birth: 'Knoxville, Tennessee, USA'
death year: null

Allow new users to register /users/new POST A JSON object holding data about the user, like so
{
username: 'valentina',
e-mail address: 'valentinavaccari@hotmail.com',
first name: 'valentina',
last name: 'vaccari',
password: 'valentina1',
favourites: [],
} 		A JSON object holding the data entered by the user, plus the user's ID
Allow users to update their user info (username) /users/[username]/[favourite movies][New-username] PATCH none A text message indicating that the new username was successfully updated
Allow users to add a movie to their list of favorites (showing only a text that a movie has been added) /users/[username]/[favourites]/[new-movietitle] POST none A text message indicating whether the movie was successfully added to the list of favourites
Allow users to remove a movie from their list of favorites (showing only a text that a movie has been removed) DELETE /users/[username]/[favourites]/[movietitle-to-remove] A text message indicating whether the movie was successfully removed from the list of favourites
Allow existing users to deregister (showing only a text that a user email has been removed) /users/[username] DELETE none A text message confirming that the user email was deregistered
